Inriawrapper 1.0.0 - Documentation¶

Inriawrapper is a Python library for to make maximum use of Sanskrit related tools available at http://sanskrit.inria.fr/. It gives facilities to decline a word in Sanskrit from its root with given parameters.

Quick Start¶

The following are the major functions offered by this module : 

  1. s(noun.gender.case.vacana)
    returns subanta form of 'noun' which has 'gender' gender, 'case' case and 'vacana' vacana. e.g. s('Davala.m.1.1') generates धवलः
  2. v(verb.gana.pada.lakara.vacya.purusa.vacana)
    returns tiGanta form of 'verb' which belongs to 'gana' gana, has 'pada' pada, is in 'lakara' tense, has 'vacya' voice, 'purusa' person, and 'vacana' vacana. e.g. v('gam.1.p.law.t.1.1') generates गच्छति
  3. k(verb.gana.kridanta.gender.case.vacana)
    returns kridanta form of 'verb' which belongs to 'gana' gana, is in 'kridanta' form, has 'gender' gender, 'case' case, and 'vacana' vacana. e.g. k('gam.1.4.m.3.1') generates गच्छमानेन
  4. adv(adverb.adv)
    returns adverbial form of 'adverb' e.g. adv('Jawiti.adv') generates झटिति.
  5. d(text)
    returns subanta / tiGanta / kridanta / adverbial form from a given word, proivded the 'text' is in proper SanskritMark specifications. e.g. d('Davala.m.1.1') generates धवलः and v('gam.1.p.law.t.1.1') generates गच्छति. the user doesn't have to bother about selecting 's', 'v', 'k' or 'adv' functions. The choice is made by machine in this usage.
  6. dc(inputfile,outputfile)
    dc (dictionary converter) converts a file of SanskritMark words into Devanagari output file. The input file must have only one word in each line.
  7. sdc(inputfile,outputfile)
    sdc (Sanskritmark-> Devanagari converter) function converts document written in SanskritMark to Devanagari output.
  8. wtd(devanagariword)
    wtd (word type decider) returns the type of input word like Noun, Verb etc e.g. wtd(धवलः) returns 'Noun' wtd(गच्छति) returns 'Verb'.
  9. r(devanagariword)
    returns SanskritMark equivalent of given Devanagari word e.g. r('धवलः') returns 'Davala.m.1.1' and r('गच्छति') returns 'gam.1.p.law.t.1.1'.
  10. dsc(inputfile,outputfile)
    dsc(Devanagari -> SanskritMark converter) function converts document written in Devanagari to SanskritMark.

You can import the module and use it in python shell or your python file as shown below:

import inriawrapper as i
print i.d('Davala.m.1.1')
#धवलः

Functions¶

The following subsections would show the explanation of different parameters of various functions.

s(noun.gender.case.vacana)¶

'gender' takes 'm'/'f'/'n'/'a' for musculine, feminine, neuter and any gender.
'case' takes '1'/'2'/'3'/'4'/'5'/'6'/'7'/'0' for nominative/accusative/instrumental/dative/ablative/genitive/locative and sambodhana respectively.
'vacana' takes '1'/'2'/'3' for ekavacana, dvivacana and bahuvacana respectively

e.g. s('Davala.m.1.1')

N.B. For noun, pronoun and vocative derivation, this function is useful.

v(verb.gana.pada.lakara.vacya.purusa.vacana)¶

'gana' takes '1' to '10' where they are usual gaNas in pANini's grammar. Use '0' for secondary verbs.
'pada' takes 'p'/'a' for parasmaipada and Atmanepada respectively.
'lakara' takes 'law'/'laN'/'viliN'/'low'/'lfw'/'lfN'/'luw'/'liw'/'luN'/'aluN'/'AliN'. viliN stands for viDiliN. aluN stands for AgamAbhAvayuktaluN, AliN stands for ASIrliN. All others have their usual notations in pANini's grammar.
'vacya' takes 't' / 'm' for kartari and karmaNi respectively.
'purusa' takes 'p'/'m'/'u' for prathama, madhyama, uttama respectively.
'vacana' takes '1'/'2'/'3' for ekavacana, dvivacana and bahuvacana respectively.

e.g. v('gam.1.p.law.t.1.1')

adv(adverb.adv)¶

'advreb' is the adverb that is to be declined.
'adv' is a constant string.

e.g. adv('Jawiti.adv')

k(verb.gana.kridanta.gender.case.vacana)¶

'gana' takes '1' to '10' where they are usual gaNas in pANini's grammar. Use '0' for secondary verbs.
# 'kridanta' takes '1' for 'kta', '2' for 'ktavat', '3' for 'Satf', '4' for 'SAnac', '5' for 'SAnac Atmanepada', '6' for 'luwAdeSa parasmaipada', '7' for 'luwAdeSa Atmanepada', '8' for 'tavya', '9' for 'yat', '10' for 'anIyar', '11' for 'Ryat', '12' for 'liqparasmai' and '13' for 'liqAtmane'.
'gender' takes 'm' for musculine, 'f' for feminine and 'n' for neuter.
'case' takes '1' for nominative, '2' for accusative, '3' for instrumental, '4' for dative, '5' for ablative, '6' for genitive, '7' for locative and '0' for sambodhana.
'vacana' takes '1' for ekavacana, '2' for dvivacana and '3' for bahuvacana respectively.

e.g. k('gam.1.4.m.1.1')

d(text)¶

function 'd' stands for declention. It decides the proper declention function to chose i.e. 's', 'v', 'k' or 'adv'.
The 'text' should be in proper SanskritMark specifications e.g. 'Davala.m.1.1' or 'gam.1.p.law.t.1.1'

e.g. d('Davala.m.1.1')

dc(inputfile, outputfile)¶

This function converts a file of SanskritMark words into Devanagari output file.
The input file must have only one word in each line.

e.g. dc('input.txt', 'output.txt')

sdc(inputfile, outputfile)¶

Function sdc(Sanskritmark-> Devanagari converter) converts document written in SanskritMark to Devanagari output.
The input should be stored in a text file. words must be separated by space.

e.g. sdc('input.txt', 'output.txt')

wtd(devanagariword)¶

Function wtd (word type decider) decides the type of input word e.g. Noun, Verb etc.
Possible output are ['Verb', 'Noun', 'Pron', 'Part', 'Advb', 'Abso', 'Voca', 'Iic', 'Ifc', 'Iiv', 'Piic']
For full form of abbreviations see this document.

e.g. wtd('धवलः')

r(devanagariword)¶

Function 'r' converts a word from Devanagari to SanskritMark.
For verbs, the expected output format is verb.gana.pada.lakara.vacya.purusa.vacana
For nouns, the expected output format is noun.gender.case.vacana
If there are more than one parsing possible, the expected output is separated by '|'.

e.g. r('धवलः')

dsc(inputfile,outputfile)¶

function dsc (Devanagari -> SanskritMark converter) to convert document written in Devanagari to SanskritMark.
The input should be stored in a file.
Words must be separated by space.

e.g. dsc('input.txt', 'output.txt')
